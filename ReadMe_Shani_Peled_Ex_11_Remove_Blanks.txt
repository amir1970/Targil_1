Project Name: String Manipulation - Remove Extra Blanks
Student Name: Shani Peled
Exercise: Ex 11, Question 1

---------------------------------------------------------
1. DESCRIPTION
---------------------------------------------------------
This program is written in ANSI C and is designed to process 
a string input by removing consecutive blank spaces. It 
ensures that only a single space remains between words, 
effectively cleaning up the text formatting.

---------------------------------------------------------
2. FILES IN THIS PROJECT
---------------------------------------------------------
* Shani_Peled_remove_blanks.c: Core logic for string parsing 
  and space removal.
* Shani_Peled_remove_blanks.h: Header file containing 
  function prototypes.
* Makefile: Automation script for compiling the project 
  with strict ANSI flags (-ansi -Wall -pedantic).

---------------------------------------------------------
3. COMPILATION AND EXECUTION
---------------------------------------------------------
To compile and run the program, use the following commands 
in the terminal (ensure you are inside the Targil_1 folder):

1. To Compile:
   make

2. To Run:
   ./Shani_Peled_remove_blanks.exe

3. To Clean Build Files:
   make clean
